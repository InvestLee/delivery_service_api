# delivery_service_api
배달 서비스 Api 프로젝트

[기능]
- 사용자 회원가입 및 로그인 기능
- 가맹점 등록, 목록 검색 기능
- 메뉴 등록, 메뉴 검색 기능
- 주문, 현재 주문 상태, 주문 이력 기능
- 접수, 접수 내역 기능
